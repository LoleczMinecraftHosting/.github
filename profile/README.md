# Lolecz Minecraft Hosting
Small project of mine for hosting and managing Minecraft servers through Discord *(And in futureTM a Web interface)*

## Can I host servers for you?
If you want feel free to contact me at minecraft@loleczkowo.com and I'll can see what I can do.  

## Repos and nerdy stuff
[`Core`](https://github.com/LoleczMinecraftHosting/Core) - Takes care of discord and managing all the servers. Also supports hosting on the server.

I run the minecraft servers in screen's so they can be accessed and wont close when the script resets.  
This project also uses an edited version of the [Discord Multicog Template](https://github.com/loleczkowo/discord-multicog-template) to work on multiple servers.

## Please note
This is not meant to be a big hosting service.
It is mostly a personal project for my own and friends servers.
